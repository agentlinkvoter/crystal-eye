# Crystal Eye

A modern phishing simulation toolkit for cybersecurity professionals.

## Installation

```bash
uv sync
uv run crystal-eye
```

## Usage

Launch the REPL and use the interactive setup wizard:

```
crystal-eye > setup
```

Or configure manually:

```
crystal-eye > set campaign my-pentest
crystal-eye > set template facebook
crystal-eye > start
```
